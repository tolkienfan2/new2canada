# Healthcare Wait Periods for Newcomers

| Jurisdiction | Wait Period |
| :--- | :--- |
| **Ontario (ON)** | None (Immediate eligibility upon arrival/application) |
| **Quebec (QC)** | Up to 3 months |
| **British Columbia (BC)** | Balance of month + 2 months (up to 3 months) |
| **Alberta (AB)** | None (Immediate eligibility upon application) |
| **Manitoba (MB)** | 3 months |
| **Saskatchewan (SK)** | 3 months |
| **Nova Scotia (NS)** | Up to 3 months |
| **New Brunswick (NB)** | Up to 3 months |
| **Newfoundland & Labrador (NL)** | None (Immediate eligibility upon arrival/application) |
| **Prince Edward Island (PEI)** | 3 months |
| **Yukon (YT)** | 3 months |
| **Northwest Territories (NT)** | None (Immediate eligibility upon application) |
| **Nunavut (NU)** | None (Immediate eligibility upon application) |

> [!IMPORTANT]
> **Key Takeaway**: If your province has a wait period (3 months max), **purchase private insurance for the interim.**
